# 项目简介

## 核心功能
- 用户管理
- 商品类型管理
- 校内信息共享管理
- 订单信息管理
- 标签管理
- 资讯文章管理
- 系统管理
- 个人中心

## 适用场景
- 校园信息共享
- 校内资讯发布
- 用户互动交流

## 技术特点
- 基于SpringBoot和Vue的B/S架构
- MVC模式
- 使用Java、Maven、MySQL等技术开发
- 采用协同过滤算法进行个性化推荐
- 支持Echarts可视化展示
- 提供详尽的文档和视频教程
